Oculus Geometry

Instrumento visual interactivo y contemplativo


---

Descripción

Oculus Geometry es un instrumento visual interactivo orientado a la exploración de geometría, tiempo y movimiento matemático. Funciona como una pieza perceptual viva: un espacio donde formas, órbitas y relaciones numéricas se despliegan sin narrativa impuesta, invitando a la observación prolongada, la pausa y la interpretación personal.

El sistema no busca representar fenómenos físicos ni optimizar resultados visuales inmediatos. Prioriza, en cambio, la coherencia estructural, la reproducibilidad temporal y la experiencia contemplativa.

> English version available: README.en.md




---

Alcance del repositorio

Este repositorio corresponde a una versión demostrativa y abierta del proyecto. Su finalidad es:

Documentar el comportamiento visual del sistema

Compartir una experiencia geométrica interactiva

Servir como base experimental y contemplativa


El núcleo conceptual y evolutivo de Oculus Geometry forma parte de una línea de investigación más amplia y no se encuentra expuesto en este repositorio.

Este repositorio representa un límite consciente, no el instrumento completo.


---

Concepto

Oculus Geometry explora:

Relaciones geométricas impulsadas por constantes matemáticas

El tiempo como parámetro escalable y perceptivo

Coherencia visual mediante interacción mínima

Reproducibilidad a largo plazo por sobre efectos inmediatos


El proyecto prioriza la claridad estructural por sobre la acumulación de funciones.


---

Estado del proyecto

Demo pública: disponible

Núcleo del sistema: intencionalmente ausente

Arquitectura: pensada para crecimiento a largo plazo
