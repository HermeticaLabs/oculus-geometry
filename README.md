# Oculus Geometry

**Oculus Geometry** es un instrumento visual interactivo y contemplativo enfocado en geometría, tiempo y movimiento matemático.

Está concebido como un sistema mínimo: sin narrativa, sin gamificación, sin significados externos impuestos — solo estructura, ritmo y forma.

Este repositorio contiene una **versión demo pública**, destinada a visualización, ejecución y estudio a nivel conceptual.

---

## Concepto

Oculus Geometry explora:

- Relaciones geométricas impulsadas por constantes matemáticas  
- El tiempo como parámetro escalable y perceptivo  
- Coherencia visual mediante interacción mínima  
- Reproducibilidad a largo plazo por sobre efectos inmediatos  

El proyecto prioriza la claridad estructural por sobre la acumulación de funciones.

---

## Estado

- Demo pública: **disponible**
- Núcleo del sistema: **intencionalmente ausente**

Este repositorio representa un **límite**, no el instrumento completo.
